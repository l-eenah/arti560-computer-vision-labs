# ARTI 560 Computer Vision Laboratory 

## Overview
This repository contains code, datasets, and exercises for various **computer vision tasks** using **TensorFlow** and **Keras**. The labs are designed to teach practical concepts in computer vision, including image classification, object detection, segmentation, and more.

## Objectives
- Learn fundamental and advanced computer vision techniques.
- Work with pretrained models and transfer learning.
- Explore model architectures, performance, and optimization.
- Apply computer vision methods to real-world datasets.

## Installation Using Conda

1. **Fork the repository**
   - Open this repository on GitHub.
   - Click the **Fork** button (top-right corner).
   - This will create your own copy of the repository under your GitHub account.

2. **Clone your forked repository**
   - Go to your forked version of the repository on GitHub.
   - Click **Code** â†’ copy the URL.
   - Open Git Bash and then run:
```
git clone <your_forked_repository_url> <directory-name>
cd <repository_folder>
```
3. **Create a Conda environment**
Replace the python version with the version currently installed on your device.
```
conda create -n cv_lab python=3.13.0
```

4. **Activate the environment**
```
source activate base
conda activate cv_lab
```

5. **Install required packages**

```
pip install -r requirements.txt
```
